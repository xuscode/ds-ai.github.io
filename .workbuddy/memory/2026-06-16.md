## 2026-06-16

### MCP 面板改造 — V5/V6 双配置块
- 原来的 MCP 面板只有一个合并的 `mcp.json` 代码块，拆分为左右并排的两个独立代码块：
  - 左侧：**CATIA V5**（本地 Python stdio 方式，`catia-v5` server）
  - 右侧：**3DEXPERIENCE V6**（streamable HTTP 方式，`3dexperience` server）
- 各自有独立的复制按钮，可单独复制对应的 JSON 配置
- 新增 `mcp.json` 根目录文件，包含 V5 + V6 的完整合并配置供下载/导入使用
- CSS 新增 `.mcp-grid` 样式，桌面端并排（1fr 1fr），移动端堆叠（1fr）

### 工具文档页重构
- 侧边栏从 4 组（零件/装配/工程图/PLM）改为 2 组：**基础** (4) + **几何** (5)
- 全部 9 个工具替换为实际 API：catia_connect, catia_status, new_part, save_current_document, create_point, create_sphere_surface, create_cylinder_surface, create_offset_plane, new_part_with_sphere
- 描述改为"覆盖 CATIA V5 基础连接、零件创建及几何曲面建模"
- create_sphere_surface 和 new_part_with_sphere 加了 ★ 角标
- 新增 .tool-star CSS 样式

### 代码块宽度调整
- `.code-block-wrapper` max-width 从 700px → 880px，CLI 和单块 MCP 面板更宽

### CLI 代码块更新
- CLI 面板的安装方式从 `pip install` 改为 PowerShell 一键安装 `iex(irm ...)`
- 支持 5 个版本：V6 (2025/2026)、V5 (R27/R29/R36)
- 步骤文案同步更新，标签改为 PowerShell

### GitHub 推送
- 仓库 `xuscode/ds-ai.github.io`（HTTPS remote `github`），本地 `master` 分支，目标分支 `main`
- 之前通过 API 推送了 3 个文件，剩余文件用户选择手动推送
