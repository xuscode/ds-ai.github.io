## 2026-06-16

### MCP 面板改造 — V5/V6 双配置块
- 原来的 MCP 面板只有一个合并的 `mcp.json` 代码块，拆分为左右并排的两个独立代码块：
  - 左侧：**CATIA V5**（本地 Python stdio 方式，`catia-v5` server）
  - 右侧：**3DEXPERIENCE V6**（streamable HTTP 方式，`3dexperience` server）
- 各自有独立的复制按钮，可单独复制对应的 JSON 配置
- 新增 `mcp.json` 根目录文件，包含 V5 + V6 的完整合并配置供下载/导入使用
- CSS 新增 `.mcp-grid` 样式，桌面端并排（1fr 1fr），移动端堆叠（1fr）

### 代码块宽度调整
- `.code-block-wrapper` max-width 从 700px → 880px，CLI 和单块 MCP 面板更宽

### CLI 代码块更新
- CLI 面板的安装方式从 `pip install` 改为 PowerShell 一键安装 `iex(irm ...)`
- 支持 5 个版本：V6 (2025/2026)、V5 (R27/R29/R36)
- 步骤文案同步更新，标签改为 PowerShell

### GitHub 推送
