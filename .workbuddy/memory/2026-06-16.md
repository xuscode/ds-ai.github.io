### MCP 面板改造 — V5/V6 双配置块
- 原来的 MCP 面板只有一个合并的 `mcp.json` 代码块，拆分为左右并排的两个独立代码块：
  - 左侧：**CATIA V5**（本地 Python stdio 方式，`catia-v5` server）

- 各自有独立的复制按钮，可单独复制对应的 JSON 配置
- 新增 `mcp.json` 根目录文件，包含 V5 + V6 的完整合并配置供下载/导入使用
- CSS 新增 `.mcp-grid` 样式，桌面端并排（1fr 1fr），移动端堆叠（1fr）

### copyCode 函数修复（2026-06-18）
- 问题：V6 代码块包含 HTML 实体（`&lt;YOUR_3DS_TOKEN&gt;`），`innerText` 可能在某些浏览器/本地文件协议下返回异常
- 修复：改用临时 `<textarea>` 解码 HTML 实体，并先剥离 `<span>` 高亮标签，再赋给 `textarea.innerHTML`，通过 `textarea.value` 获取纯文本
- 文件：`js/main.js`（`copyCode` 函数）

### 页眉页脚组件化（js/components.js）

- 所有页面 nav/footer 替换为 `<div id="nav-placeholder"></div>` 和 `<div id="footer-placeholder"></div>`
