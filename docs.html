工具文档

共 90 个工具，覆盖 CATIA V5 基础连接、几何曲面、草图、Part Design 及文档管理五大模块，

目前AI可以落地的,帮助工程师完成一些自动化和批量化的任务,例如实例化超级副本等模板,导出工程数据,数据的输入输出
在生成设计图纸这一项中，自动创建工程图框，插入2D部件，一些固定化的标注和注释生成，可以做一些批量的标注属性修改，
因为接口限制，虽然有很多不完善的地方，是经过日积月累，会为我们的工作带来很大的便利性

CATIA V5

catia_connect

建立与 CATIA V5 应用程序的连接会话

入参

参数名类型必填说明
hoststring—CATIA 主机地址，默认 localhost
portnumber—CATIA 通信端口
例子
# 连接本地 CATIA
{
  "host": "localhost",
  "port": 8099
}

响应 data

字段类型说明
sessionIdstring会话唯一标识
catiaVersionstringCATIA 版本号
statusstring连接状态：connected
