CATIA V5 — 零件

partCreate

在 CATIA V5 中创建新零件文档，并支持批量添加建模特征

入参

参数名类型必填说明
partNamestring必填零件名称，保存 .CATPart 文件名
featuresarray[object]—特征列表，按顺序创建。每项包含以下字段
typestring必填特征类型：pad / pocket / hole / shaft / groove / rib / slot / fillet / chamfer / draft / shell / mirror
sketchobject必填草图定义。包含以下字段
planestring—草图平面：xy / yz / zx，默认 xy
profilearray[array[number]]必填轮廓点坐标列表 [[x1,y1], [x2,y2], ...] 毫米
limitnumber | string必填拉伸长度 (mm) 或终止条件：upToNext / upToLast / upToPlane / upToSurface
directionstring—拉伸方向：positive / negative / symmetric，默认 positive
materialstring—材料名称，如 Steel / Aluminum / Titanium
例子
# 创建一个带四个沉头孔的铝合金法兰盘
{
  "partName": "Flange_Plate_001",
  "material": "Aluminum",
  "features": [
    { "type": "pad", "sketch": { "plane": "xy", "profile": [[0,0],[200,0],[200,200],[0,200]] }, "limit": 25 },
    { "type": "hole", "position": [30, 30, 0], "diameter": 10, "depth": 25, "holeType": "countersunk", "headDiameter": 16 }
  ]
}

响应 data

字段类型说明
partIdstring零件唯一 ID
partNamestring零件名称
filePathstring保存路径
boundsobject包围盒信息
lengthnumberX 方向长度 mm
widthnumberY 方向宽度 mm
heightnumberZ 方向高度 mm
featureCountnumber创建的特征数量
featuresarray[object]创建的特征列表
featureIdstring特征唯一 ID
typestring特征类型
statusstring创建状态：ok / warning / failed
CATIA V5 — 零件

partModifyParam

修改已打开零件的特征参数、草图和约束

入参

参数名类型必填说明
partIdstring必填目标零件 ID，传入 active 可操作当前活动零件
featureIdstring必填要修改的特征 ID
paramsobject必填要修改的参数键值对
limitnumber—修改拉伸/切除长度 mm
diameternumber—修改孔径 mm
radiusnumber—修改圆角/倒角半径 mm
anglenumber—修改角度 °
recomputeboolean—是否执行 Part.Update 刷新模型，默认 true
例子
# 将凸台拉伸长度改为 50mm，圆角半径改为 3mm，并刷新
{
  "partId": "active",
  "featureId": "Pad.1",
  "params": { "limit": 50, "radius": 3 },
  "recompute": true
}

响应 data

字段类型说明
featureIdstring修改的特征 ID
updatedParamsobject成功更新的参数键值对
updateStatusstring刷新结果：ok / needsFix / failed
warningsarray[string]刷新警告信息列表
CATIA V5 — 零件

partExport

将零件导出为 STEP、IGES、STL 等中性格式文件

入参

参数名类型必填说明
partIdstring必填零件 ID 或 active
formatstring必填导出格式：step / iges / stl / catpart
outputPathstring必填输出路径，支持绝对路径或相对路径
optionsobject—格式相关选项
stepVersionstring—STEP 版本：AP203 / AP214 / AP242，默认 AP214
stlResolutionnumber—STL 网格精度 0-1，默认 0.01
stlBinaryboolean—STL 是否二进制格式，默认 true
例子
# 导出当前活动零件为 STEP AP242 格式
{
  "partId": "active",
  "format": "step",
  "outputPath": "D:/Export/Bracket_001.stp",
  "options": { "stepVersion": "AP242" }
}

响应 data

字段类型说明
outputPathstring实际输出文件完整路径
formatstring导出格式
fileSizenumber文件大小 bytes
durationnumber导出耗时 ms
CATIA V5 — 装配

assemblyLoad

在 CATIA 中加载装配体，支持批量插入子组件和零件，并返回装配树结构

入参

参数名类型必填说明
assemblyPathstring—已有 .CATProduct 路径，为空则新建装配
assemblyNamestring—新装配体名称，默认 Assembly_001
componentsarray[object]—要插入的零件/子装配列表
filePathstring必填零件或子装配文件路径
positionarray[number]—插入坐标 [x, y, z] mm，默认 [0, 0, 0]
instanceNamestring—实例名称，默认用文件名
例子
# 新建装配体并插入三个零件
{
  "assemblyName": "Gearbox_Main",
  "components": [
    { "filePath": "D:/Parts/Housing.CATPart", "position": [0, 0, 0] },
    { "filePath": "D:/Parts/Gear_Input.CATPart", "position": [50, 0, 0] },
    { "filePath": "D:/Parts/Gear_Output.CATPart", "position": [50, 120, 0] }
  ]
}

响应 data

字段类型说明
assemblyIdstring装配体 ID
assemblyNamestring装配体名称
componentCountnumber总组件数量
treearray[object]装配树结构
instanceIdstring实例唯一 ID
instanceNamestring实例名称
partNumberstring零件编号（PartNumber）
positionarray[number]当前位置 [x, y, z]
childrenarray[object]子组件列表（递归结构）
CATIA V5 — 装配

assemblyAddConstraint

在装配体中为两个组件添加几何约束（重合、接触、偏移、角度）

入参

参数名类型必填说明
assemblyIdstring必填装配体 ID
constraintTypestring必填约束类型：coincidence / contact / offset / angle / fix
refElementAobject必填第一个参考元素
instanceIdstring必填组件实例 ID
elementstring必填几何元素标识，如 xyPlane / zxFace / zxPlane / cylindricalFace
refElementBobject必填第二个参考元素（结构同上）
offsetnumber—偏移距离 mm（offset 类型必填）
anglenumber—角度值 °（angle 类型必填）
orientationstring—方向：same / opposite，默认 same
例子
# 将两个零件的 XY 平面设为重合约束
{
  "assemblyId": "Product.1",
  "constraintType": "coincidence",
  "refElementA": { "instanceId": "Gear_Input.1", "element": "xyPlane" },
  "refElementB": { "instanceId": "Gear_Output.1", "element": "xyPlane" },
  "orientation": "opposite"
}

响应 data

字段类型说明
constraintIdstring创建的约束 ID
constraintTypestring约束类型
statusstring状态：ok / overconstrained / invalid
dofRemainingnumber剩余自由度
工程图

drawingGenerate

从零件或装配体自动生成工程图，支持多视图、标注和 BOM 表

入参

参数名类型必填说明
sourceIdstring必填零件 ID 或装配体 ID
viewsarray[object]必填视图定义列表
typestring必填视图类型：front / top / right / isometric / section / detail / auxiliary
positionarray[number]必填视图在图框位置 [x, y] mm
scalenumber—视图比例，如 1.0, 2.0, 0.5，默认 1.0
hiddenLinesboolean—是否显示隐藏线，默认 false
sheetSizestring—图幅：A0 / A1 / A2 / A3 / A4，默认 A3
standardstring—制图标准：ISO / ANSI / JIS / GB，默认 ISO
includeBOMboolean—装配图是否生成 BOM 表，默认 false
dimensionsboolean—是否自动标注尺寸，默认 true
例子
# 生成 A3 ISO 工程图：主视图 + 右视图 + 等轴测，自动标注
{
  "sourceId": "Part.1",
  "sheetSize": "A3",
  "standard": "ISO",
  "dimensions": true,
  "views": [
    { "type": "front", "position": [120, 150], "scale": 1.0 },
    { "type": "right", "position": [320, 150], "scale": 1.0 },
    { "type": "isometric", "position": [420, 300], "scale": 0.5 }
  ]
}

响应 data

字段类型说明
drawingIdstring工程图文档 ID
sheetCountnumber图页数量
viewsarray[object]创建的视图列表
viewIdstring视图 ID
typestring视图类型
statusstring生成状态：ok / needsUpdate / failed

3DEXPERIENCE — PLM

plmSearch

在 3DEXPERIENCE PLM 平台中搜索工程数据（零件、装配、图纸、变更单等）

入参

参数名类型必填说明
querystring必填搜索关键字，支持 Part Number / Title / Description
objectTypestring—对象类型：Part / Assembly / Drawing / ECO / MCO / Document，默认全部
statestring—生命周期状态：InWork / Frozen / Released / Obsolete
ownerstring—负责人/创建者用户名
pageSizenumber—每页返回数量，默认 20
pagenumber—页码，从 1 开始，默认 1
例子
# 搜索所有已发布的支架类零件
{
  "query": "Bracket",
  "objectType": "Part",
  "state": "Released",
  "pageSize": 10
}

响应 data

字段类型说明
totalResultsnumber总结果数
pagenumber当前页码
resultsarray[object]查询结果列表
objectIdstring对象 PLM ID
typestring对象类型
partNumberstring零件编号
titlestring标题
revisionstring当前版次
statestring生命周期状态
modifiedstring最后修改时间 ISO 8601
3DEXPERIENCE — PLM

plmGetBOM

获取指定对象的 BOM 结构（物料清单），支持多层级展开

入参

参数名类型必填说明
objectIdstring必填根对象 PLM ID（装配或产品）
depthnumber—展开层级深度，1 为仅第一层，-1 为全部展开，默认 -1
configstring—配置/变型筛选，如 Design / Manufacturing
includeQuantityboolean—是否包含数量，默认 true
includeMetadataboolean—是否包含元数据（材料/重量等），默认 true
例子
# 获取装配体全部 BOM 结构，包含材料与数量
{
  "objectId": "PLM-2024-GB-00123",
  "depth": -1,
  "includeQuantity": true,
  "includeMetadata": true
}

响应 data

字段类型说明
rootObjectIdstring根对象 ID
bomItemsarray[object]BOM 行项目列表
levelnumber层级（1 为直接子件）
objectIdstring子件 PLM ID
partNumberstring零件编号
titlestring标题
quantitynumber数量
unitstring单位：ea / kg / m / L
materialstring材料
massnumber质量 kg
childrenarray[object]子项 BOM（递归结构）
totalItemsnumberBOM 总行数（含所有层级）
totalMassnumber总质量 kg
