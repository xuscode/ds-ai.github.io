CATIA V5 — 基础

catia_connect

建立与 CATIA V5 应用程序的连接会话

入参

参数名类型必填说明
hoststring—CATIA 主机地址，默认 localhost
portnumber—CATIA 通信端口
例子
# 连接本地 CATIA
{
  "host": "localhost",
  "port": 8099
}

响应 data

字段类型说明
sessionIdstring会话唯一标识
catiaVersionstringCATIA 版本号
statusstring连接状态：connected

CATIA V5 — 基础

save_current_document

保存当前 CATIA 活动文档到指定路径

入参

参数名类型必填说明
pathstring—保存路径。为空则覆盖当前文件
例子
# 另存为指定路径
{
  "path": "D:/Parts/MyPart.CATPart"
}

响应 data

字段类型说明
savedPathstring实际保存路径
documentNamestring文档名称
CATIA V5 — 几何

create_point

在活动零件中创建一个参考点，支持绝对坐标和偏移方式

入参

参数名类型必填说明
xnumber必填X 坐标 mm
ynumber必填Y 坐标 mm
znumber必填Z 坐标 mm
例子
# 在 (100, 50, 0) 位置创建点
{
  "x": 100,
  "y": 50,
  "z": 0
}

响应 data

字段类型说明
pointIdstring点对象 ID
positionobject实际坐标 {x, y, z}
CATIA V5 — 几何

create_sphere_surface ★

在活动零件中创建球体曲面，支持指定球心和半径

入参

参数名类型必填说明
centerobject—球心坐标。为空则默认原点 {x:0, y:0, z:0}
xnumber—球心 X 坐标 mm
ynumber—球心 Y 坐标 mm
znumber—球心 Z 坐标 mm
radiusnumber必填球体半径 mm
例子
# 在原点创建半径 50mm 的球体曲面
{
  "radius": 50
}

响应 data

字段类型说明
surfaceIdstring曲面对象 ID
radiusnumber球体半径 mm
centerobject球心坐标 {x, y, z}
CATIA V5 — 几何

create_cylinder_surface

在活动零件中创建圆柱曲面，支持指定圆心轴、半径和高度

入参

参数名类型必填说明
radiusnumber必填圆柱半径 mm
heightnumber必填圆柱高度 mm
centerobject—底面圆心坐标，默认 {x:0, y:0, z:0}
axisstring—轴线方向：x / y / z，默认 z
例子
# 创建半径 30mm、高 80mm、Z 轴圆柱曲面
{
  "radius": 30,
  "height": 80,
  "axis": "z"
}

响应 data

字段类型说明
surfaceIdstring曲面对象 ID
radiusnumber圆柱半径 mm
heightnumber圆柱高度 mm
CATIA V5 — 几何

create_offset_plane

在活动零件中创建偏移参考平面

入参

参数名类型必填说明
referencestring必填参考平面：xy / yz / zx
offsetnumber必填偏移距离 mm，可为负值
例子
# 从 XY 平面偏移 50mm 创建新平面
{
  "reference": "xy",
  "offset": 50
}

响应 data

字段类型说明
planeIdstring平面对象 ID
referencestring参考平面
offsetnumber偏移距离 mm

new_part_with_sphere ★

一键创建新零件并生成球体曲面，是最快捷的建模入口

入参

参数名类型必填说明
partNamestring—零件名称，默认 Part1

  "radius": 100
}

响应 data

partNamestring零件名称
surfaceIdstring球体曲面 ID
radiusnumber球体半径 mm
