自然语言建模

打开 CATIA，用自然语言描述设计意图，AI 自动执行建模操作

CATIA V5 — mcp.json 
{
  "mcpServers": {
    "catia-v5": {
      "command": "python",
      "args": ["-m", "catia_v5_mcp.server"],
      "env": {
        "CATIA_HOST": "localhost",
        "CATIA_PORT": "8099"
      }
    }
  }
}
3DEXPERIENCE V6 — mcp.json 
{
  "mcpServers": {
    "3dexperience": {
      "command": "python",
      "args": ["-m", "3dexperience_mcp.server"],
      "env": {
        "3DS_URL": "https://3ds.example.com",
        "3DS_TOKEN": "<YOUR_3DS_TOKEN>"
      }
    }
  }
}
